# 📁 Folder Cleaner - AI-Powered File Organization Tool

A professional, scalable file organization tool built with Python. Automatically sorts files into folders based on their type, extension, or custom rules.

## 🎯 Features

- ✅ **Smart File Detection** - Automatically identifies file types (images, documents, archives, etc.)
- ✅ **Multiple Organization Modes**
  - By file extension (.pdf, .jpg, .txt, etc.)
  - By file type (Images, Documents, Videos, etc.)
  - Custom rules (your own categories)
- ✅ **User-Friendly GUI** - Built with Tkinter
- ✅ **Safe Operations** - Preview before moving, error handling
- ✅ **Scalable** - Handles thousands of files efficiently
- ✅ **Cross-Platform** - Works on Windows, Mac, and Linux

## 📋 Requirements

- Python 3.8 or higher
- No external dependencies (uses only Python standard library)
